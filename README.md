# Gulp
Install

1) Install <a href="https://nodejs.org/">NodeJS</a>
2) Install Gulp <code>npm install gulp -g</code>

3) Create <code>package.json</code>:

```
{
    "name": "Application",
    "version": "0.0.1"
}
```

4) Create <code>gulpfile.js</code>:

```
  var gulp = require('gulp');
  
  gulp.task('default', []); 
```

5) Run <code>gulp</code>.
